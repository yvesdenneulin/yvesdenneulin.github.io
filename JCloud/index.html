Manipulations de l'après-midi

Vous devez avoir d'abord ouvert votre compte temporaire AWS en répondant au mail reçu. Suivez ensuite attentivement la documentation pour démarrer votre laboratoire AWS qui vous donne accès à une version partielle de la console AWS. Cette dernière s'ouvre dans une fenêtre qui peut avoir été filtrée par votre gestionnaire de pop-ups.

Une fois que vous avez accédé à la console vous pouvez essayer de réaliser les opérations suivantes :

  1. créer un bucket avec S3
  2. ajouter un fichier quelconque dans ce bucket
  3. créer une machine virtuelle EC2 Linux
    1. se connecter dessus avec ssh
    2. récupérer et afficher le fichier uploadé
    3. bonus : explorer la machine créée, en essayer d’autres (/proc/cpuinfo, /proc/meminfo, df -h, …)
